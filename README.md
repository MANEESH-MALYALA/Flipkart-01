# Flipkart Clone

A Flipkart-inspired e-commerce homepage built with HTML and CSS.

## Features
- Responsive header with search bar
- Product grid
- Categories navigation
- Footer with links

## Tech
HTML5 · CSS3

## Run it
Open [index.html](http://127.0.0.1:5500/MAIN/INDEX.HTML) in any browser. No build step needed.
